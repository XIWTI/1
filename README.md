# Love

Автономное романтичное сайт-приложение без сервера. Работает как статический сайт, сохраняет прогресс в `localStorage`, играет музыку из папки `music` и подходит для публикации на GitHub Pages.

## Запуск

Откройте `index.html` в браузере или запустите локальный статический сервер из этой папки.

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

После этого сайт будет доступен по адресу:

```text
http://127.0.0.1:8765/
```

## Что внутри

- стартовый вопрос с кнопкой `Нет`, которая убегает;
- доступ к играм только после ответа `Да`;
- Memory Love с 3 уровнями, пятнашки 3x3/4x4/5x5, одноразовая викторина, шахматы с выбором стороны, 5 уровнями робота и режимом для двоих;
- hearts economy, daily tasks, достижения, магазин, награды и колесо фортуны;
- точное сохранение даты и времени `Claim Gift`;
- музыкальный плеер с локальными треками, shuffle/repeat, громкостью, переключением и выбором песни;
- промокоды: `солнышко` даёт 4000 hearts один раз, `ILU` открывает сообщения после 20000 hearts;
- звуки ходов, ошибок, совпадений и побед;
- полноэкранное открытие игр и кнопка полного сброса прогресса до вопроса Да/Нет;
- ночная атмосфера и секретное достижение вечером или ночью;
- PWA-манифест и service worker для установки на iOS/Android/PC и кэширования файлов после первого открытия.
